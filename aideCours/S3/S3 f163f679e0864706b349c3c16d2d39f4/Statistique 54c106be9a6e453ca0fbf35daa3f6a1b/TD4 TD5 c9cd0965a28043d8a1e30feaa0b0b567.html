TD4

  1. Loi de Bernoulli (un essais):
    • Utilisée pour modéliser une expérience aléatoire avec deux résultats possibles : succès (1) ou échec (0).
    • Un exemple classique est le lancer d'une pièce de monnaie une seule fois, où le succès pourrait être "pile" et l'échec "face".
    • Paramètre : p (probabilité de succès dans un essai).
  1. Loi binomiale (plusieurs essais):
    • Utilisée pour modéliser le nombre de succès dans une séquence d'essais indépendants et identiquement distribués, où chaque essai suit une loi de Bernoulli.

      avec n : Le nombre total d'essais indépendants.

      k : Le nombre de succès que l'on cherche à atteindre parmi les n essais.

    • Exemples incluent le nombre de têtes obtenues en lançant une pièce plusieurs fois, ou le nombre de succès dans un nombre fixe de tentatives indépendantes.
    • Paramètres : n (nombre total d'essais) et p (probabilité de succès dans chaque essai).
  1. Loi géométrique (avant bon resultat):
    • Utilisée pour modéliser le nombre d'essais nécessaires pour obtenir le premier succès dans une séquence d'essais indépendants de Bernoulli.
    • Exemple : le nombre de lancers d'une pièce avant d'obtenir la première fois "pile".
    • Paramètre : p (probabilité de succès dans chaque essai).
  1. Loi de Poisson (LE TEMPS) :
    • Utilisée pour modéliser le nombre d'événements rares dans un intervalle de temps ou d'espace fixe.
    • Exemples incluent le nombre d'appels reçus par une ligne téléphonique en une heure ou le nombre de particules radioactives émises en un temps donné.
    • Paramètre : λ (taux moyen d'événements rares par unité de temps ou d'espace).
  1. Loi uniforme :
    • Utilisée pour modéliser une distribution où chaque valeur dans l'intervalle possible a la même probabilité d'occurrence.
    • Exemple : la distribution uniforme entre 0 et 1, où chaque nombre réel dans cet intervalle a la même probabilité d'être choisi.
    • Paramètres : a et b (bornes de l'intervalle).

    La fonction de masse de probabilité de la distribution de Poisson est donnée par :

    P(X=k)=e−λ⋅λkk!P(X = k) = \frac{e^{-\lambda} \cdot \lambda^k}{k!}P(X=k)=k!e−λ⋅λk​﻿
