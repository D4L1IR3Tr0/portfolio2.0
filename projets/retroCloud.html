Retro Cloud Server

Introduction

Retro Cloud est un service cloud innovant développé en utilisant les technologies web telles que HTML, CSS et JavaScript (NODE.js). Il offre une interface utilisateur conviviale permettant de gérer efficacement vos fichiers en ligne, y compris les fonctionnalités de téléchargement, d'upload, de recherche et bien plus encore.

Présentation

Retro Cloud se distingue par sa simplicité et son efficacité. Il se compose d'une partie front-end et d'une partie back-end. Le front-end gère l'interface utilisateur et les interactions, tandis que le back-end enregistre les logs détaillés de toutes les activités effectuées sur le cloud.

Objectif

L'objectif principal de Retro Cloud est de fournir une solution de stockage en ligne sécurisée et conviviale pour les utilisateurs souhaitant gérer leurs fichiers de manière efficace et accessible. Que ce soit pour sauvegarder des documents importants ou pour partager des fichiers avec d'autres utilisateurs, Retro Cloud répond à ces besoins de manière simple et pratique.

Fonctionnement

Le fonctionnement de Retro Cloud est facile à comprendre :

  • Accédez à la page de connexion et connectez-vous en utilisant vos identifiants.
  • Une fois connecté, vous serez dirigé vers la page de gestion où vous pouvez :
    • Uploader des fichiers vers le cloud.
    • Télécharger des fichiers depuis le cloud.
    • Rechercher des fichiers spécifiques parmi vos données stockées.
  • Le back-end enregistre des logs détaillés de toutes les actions effectuées par les utilisateurs, garantissant ainsi une traçabilité complète des activités.
