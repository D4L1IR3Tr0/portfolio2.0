Introduction

Ce programme en Go permet d'encoder et de décoder des messages dans des fichiers MP3, avec une option de chiffrement AES-128 pour protéger le contenu du message.

Fonctionnalités

  • Encodage de message : Ajoute un message texte dans les premiers octets d'un fichier MP3.
  • Décodage de message : Extrait le message encodé dans un fichier MP3.
  • Chiffrement AES : Protégez le message encodé avec un chiffrement AES-128 en utilisant une clé de 16 caractères.

Utilisation

Usage:
-c <fichier.mp3> "message"  pour encoder un message dans un fichier MP3
-c <fichier.mp3> "message" "clé"  pour encoder un message et le crypter avec une clé de 16 caractères
-d <fichier.mp3>  pour décoder un message d'un fichier MP3
-d <fichier.mp3> "clé"  pour décoder un message crypté d'un fichier MP3

Exemples

  • -c musique.mp3 "Message secret" : Encode un message dans le fichier musique.mp3.
  • -c musique.mp3 "Message secret" "maCleDeChiffre16" : Encode et chiffre un message avec une clé.
  • -d musique.mp3 : Décodage d'un message d'un fichier MP3.
  • -d musique.mp3 "maCleDeChiffre16" : Décodage et déchiffrement d'un message protégé.

Ce programme peut être utilisé pour dissimuler des messages simples dans des fichiers audio MP3, offrant une discrétion supplémentaire avec l'ajout du chiffrement AES-128.
